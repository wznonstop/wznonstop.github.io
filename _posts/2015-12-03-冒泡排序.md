冒泡排序
===================

打算接下来的时间有空就学学算法，用JS实现，放在博客里，虽说前端用到算法的时候好像貌似似乎不多，但是学学还是好的，万一用上了呢？


	var arr = [25,32,7,14,99];
	var x = 1;
	length = arr.length;
	while(x <= (length - 1)){
		for (var i = 0; i < (arr.length-x); i++) {
			var that = arr[i];
			var cthat = arr[i+1];
			if(that<cthat){
		        var val = that;
		        arr[i] = cthat;
		        arr[i+1] = val;
		    }
		};
		x++;
	}
